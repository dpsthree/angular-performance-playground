The application

Angular Performance Playground (APP) was original created to examine the performance characteristics of Angular, D3 and SVG. Since then it has grown to encompass a varity of performance tests and proof of concepts. Ideas captured by APP include:

  • D3 integration
  • service workers
  • web workers
  • lazy loading
  • observable data flow
  • semantic use Angular Material
  • smart/view component pattern
  • trackBy function
  • OnPush change detection
  • AoT compilation

Futue aspirations include

  • render timing with animation frame
  • user controlled application of performance techniques
  • additional examples of performance demanding features
  • Layout with CSS grid and ng-flex
